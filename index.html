Free Online Dev Tools

Online tools for developers for their day to day use

This site contains client-side Javascript tools helping developers in encoding and decoding Base64 to String, Hex to String, URL encoding and decoding, formatting and validating JSON, YAML, CSV and much more!

We do not store any data in our servers as the site entirely runs on Javascript

We will continue adding new tools, so any suggestions are welcome! Feel free to send a feature request on GitHub

Source code: https://github.com/bharatmicrosystems/dev-tools
